About This Project

Learn what this tool does and how it works.

What is the Child Growth Checker?

The Child Growth Checker is a simple, browser-based tool that helps parents, teachers, and caregivers keep track of basic child growth metrics. It provides an approximate growth category using BMI and publicly available CDC/WHO guidance.

How it Works

  • You register a child with basic information (name, DOB, sex, optional notes).
  • You input height and weight to generate growth approximations.
  • The tool calculates BMI and gives a general growth category.
  • Data is stored **only locally** in your browser using localStorage.
  • No data leaves your device. Nothing is uploaded.

Important Disclaimer

This is a demonstration tool and is **not intended for clinical diagnosis or medical decision-making**. For accurate percentile-based assessments, consult healthcare professionals or use the official CDC BMI calculator, which is also embedded in this project.

Why Was This Built?

This project serves as a lightweight example of how child growth tools can be implemented entirely on the front end without storing or transmitting sensitive data. It is useful for demonstrations, school projects, parent awareness, and web development practice.

Note: This tool stores everything in your browser and does not share data with any server.
